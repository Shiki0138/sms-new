予約ウィジェット埋め込みガイド

お客様のウェブサイトに簡単に予約システムを追加できます

ライトプランの特典

  • ✓ 完全レスポンシブ対応の予約ウィジェット
  • ✓ リアルタイム空き状況確認
  • ✓ 自動確認メール送信
  • ✓ SMSリマインダー（予約前日）
  • ✓ 月間100件までの予約に対応

設定情報を入力

埋め込みコード

以下のコードをウェブサイトのHTMLに貼り付けてください。

<!-- Salon Booking Widget -->
<div id="salon-booking-widget"></div>
<script>
  window.SALON_WIDGET_CONFIG = {
    salonId: 'YOUR_SALON_ID',
    apiKey: 'YOUR_API_KEY',
    container: 'salon-booking-widget',
    lang: 'ja',
    onSuccess: function(booking) {
      console.log('Booking successful:', booking);
      // オプション: 予約成功時の処理
    },
    onError: function(error) {
      console.error('Booking error:', error);
      // オプション: エラー時の処理
    }
  };
</script>
<script src="https://cdn.salonbooking.jp/widget/v1/booking-widget.js"></script>

導入手順

管理画面でAPIキーを取得

サロン管理画面の「設定」→「API設定」からAPIキーを取得してください。

上記のフォームに情報を入力

サロンIDとAPIキーを入力すると、自動的に埋め込みコードが生成されます。

コードをウェブサイトに貼り付け

生成されたコードを、予約フォームを表示したい場所に貼り付けてください。

動作確認

ウェブサイトを更新して、予約ウィジェットが正しく表示されることを確認してください。

カスタマイズオプション

window.SALON_WIDGET_CONFIG = {
  salonId: 'YOUR_SALON_ID',
  apiKey: 'YOUR_API_KEY',
  container: 'salon-booking-widget',
  
  // 言語設定
  lang: 'ja', // 'ja' または 'en'
  
  // テーマ設定
  theme: 'light', // 'light' または 'dark'
  
  // カスタムスタイル
  customStyles: {
    primaryColor: '#4CAF50',
    fontFamily: 'Noto Sans JP'
  },
  
  // デフォルト値
  defaultService: 'service_id',
  defaultStaff: 'staff_id',
  
  // コールバック関数
  onSuccess: function(booking) {
    // 予約成功時の処理
  },
  onError: function(error) {
    // エラー時の処理
  },
  onLoad: function() {
    // ウィジェット読み込み完了時
  }
};

プレビュー

よくある質問

ウィジェットが表示されない場合

APIキーが正しいか、またサロンIDが有効か確認してください。また、ライトプラン以上のプランに加入している必要があります。

スタイルをカスタマイズしたい

CSSで .salon-booking-widget クラスをターゲットにしてスタイルを上書きできます。

複数のウィジェットを設置したい

異なるcontainer IDを指定することで、同一ページに複数のウィジェットを設置できます。
