# 美容サロン管理システム - 運用ガイド

## 🌟 システム概要

本システムは美容師・美容サロン向けに特化した総合管理システムです。
顧客管理・予約管理・LINE連携・売上管理など、サロン運営に必要な機能を一元化しています。

---

## 🚀 初回セットアップ

### 1. アカウント新規登録

1. **サイトにアクセス**
   - URL: https://salon-light-plan-h3a4l921e-shikis-projects-6e27447a.vercel.app

2. **新規登録ボタンをクリック**
   - メールアドレス
   - パスワード（8文字以上）
   - サロン名

3. **メール認証**
   - 登録したメールアドレスに認証メールが送信されます
   - リンクをクリックして認証を完了

### 2. 基本設定

**設定ページ**で以下を設定してください：

- **サロン情報**
  - サロン名
  - 住所
  - 電話番号
  - 営業時間
  - 定休日

- **スタッフ情報**
  - スタッフ名
  - 担当メニュー
  - 稼働時間

## 💼 主要機能の使い方

### 📋 1. 顧客管理

**新規顧客登録**

1. 「顧客」タブ → 「新規追加」
2. 基本情報を入力：
   - 氏名
   - 電話番号
   - メールアドレス
   - 生年月日
   - 性別
   - 住所

**顧客情報の管理**

- 来店履歴の自動記録
- 施術写真の保存
- メモ・アレルギー情報
- 満足度評価
- 次回予約提案

### 📅 2. 予約管理

**予約作成**

1. 「予約」タブ → 「新規予約」
2. 項目を選択：
   - 顧客選択
   - 施術メニュー
   - 担当スタッフ
   - 日時
   - 料金

**予約管理機能**

- カレンダー表示（日・週・月）
- 予約変更・キャンセル
- 空き時間の確認
- ダブルブッキング防止
- 自動リマインダー

### 💬 3. メッセージング・LINE連携

**LINE Bot設定**

1. 「設定」→「LINE連携」
2. LINE Developers でアカウント作成
3. Messaging API設定
4. Channel Access Token を取得・入力
5. Webhook URLを LINE に設定

**メッセージ機能**

- 顧客との個別やり取り
- AI自動返信（営業時間・予約可能時間）
- 予約確認・変更メッセージ
- 一斉配信（キャンペーン告知など）

### 📊 4. 売上・レポート機能

**売上管理**

- 日別・月別売上集計
- スタッフ別売上
- メニュー別売上分析
- 顧客単価分析
- リピート率計算

**レポート機能**

- 売上グラフ表示
- 顧客分析レポート
- 予約率・キャンセル率
- スタッフパフォーマンス

## 📱 日常の運用フロー

### 朝の準備（開店前）

1. **今日の予約確認**
   - 予約一覧で当日スケジュール確認
   - 新規予約・変更の確認

2. **メッセージ確認**
   - LINEでの問い合わせ対応
   - 予約変更・キャンセルの処理

### 施術中・施術後

1. **顧客情報更新**
   - 施術内容の記録
   - Before/After写真保存
   - 次回提案メモ

2. **売上入力**
   - 施術完了時に売上記録
   - 支払い方法の選択

### 夕方（閉店前後）

1. **売上確認**
   - 当日売上の確認
   - 月間目標との比較

2. **明日の準備**
   - 翌日予約の最終確認
   - 必要な準備物の確認

## 🛠️ トラブルシューティング

### よくある問題と解決方法

**ログインできない**

- パスワードを忘れた場合：「パスワードリセット」から再設定
- メールアドレスが間違っている：正確なアドレスで再登録

**LINE連携がうまくいかない**

- Webhook URLの確認
- Channel Access Token の再確認
- LINE Developers の設定見直し

**予約の重複**

- カレンダーで空き時間を必ず確認
- スタッフの稼働時間設定を見直し

**データが消えた**

- システムは自動保存されるため、ブラウザを更新
- ネットワーク接続を確認
